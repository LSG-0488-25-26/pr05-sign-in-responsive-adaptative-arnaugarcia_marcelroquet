# PR05 - SIGN IN RESPONSIVE AND ADAPTATIVE

By **Marcel Feliu** and **Marcel Roquet**

The aim of this project was to create a simple log in and register application to practise how responsiveness and adaptativeness as well as work in a MVVM project. There are 3 types of sizes; **Compact**, **Medium** and **Extended**.

We created a dedicated view for each size category to ensure the interface adapts seamlessly and displays information in the most user-friendly way possible across all devices. This project allowed us to practice implementing responsive and adaptive layouts within an MVVM architecture, reinforcing both design and structural principles.
